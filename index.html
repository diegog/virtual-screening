A modern drug discovery technique

Overview

Virtual screening is a computational technique used in drug discovery . It automatically evaluates very large libraries of compounds and identifies structures that are likely to bind to a target molecule . Virtual screening is a difficult process because of how enormous the chemical space is. In practice this number is reduced to the chemicals that are easy to be synthesized or purchased. Virtual Screening has two different variations: ligand-based virtual screening and structure-based virtual screening.

timeline

Structure-based

Involves the analysis of a compound in which the 3D structure is known. Estimates the binding strength between the target molecule and another molecule using a mathematical method called a scoring function.

gesture

Ligand-based

Involves the detailed analysis of active ligands to predict how similar ligands will react. This method is typically used when the 3D structure of the target molecule is unknown. Because of the uncertainty of the molecule's structure, this variation is more theoretical than structure-based variations.

NOTE: the rest of this web page will refer to techniques used in Structure-based virtual screening.

Scoring Functions

Computational methods are used to test thousands of possible interactions between a target molecule and compounds in a chemical library. The quality of the interactions are formulated using mathematical method called a scoring function to produce a score of a compound. Creating this scoring function is a crucial part of virtual screening because it directly affects the accuracy of the score. Scoring functions are commonly separated into three categories:

  • blur_onForcefield-based
    Forcefield-based scoring functions are based on the summation of protein-ligand interaction energies and internal ligand energies using well-documented molecular mechanics forcefield parameters.
  • functionsEmpirically-derived
    Derived by summing a set of individual energy terms, terms such as hydrogen bonds, hydrophobic effects, entropy, etc.
  • schoolKnowledge-based
    Energy terms are derived directly from experimentally determined structural information. Then, binding energy is calculated by adding the interaction terms of each possible protein-ligand atom pair in the entire complex using Boltzmann distributions.

There is still debate about the existence a fourth category of scoring functions that involves the use of machine learning . These debates are becoming less common as the scoring techniques are being researched. Machine learning or descriptor-based scoring functions learns patterns within structural information and then uses what it knows to make predictions about a target molecule.

Future Directions

Structure-based virtual screening seems to have a bright future ahead of it. The newest machine learning methods are now being implemented in virtual screening with great success. As new computational techniques are developed more virtual screening techniques will arise and as computers get more powerful these techniques will become more accurate.
